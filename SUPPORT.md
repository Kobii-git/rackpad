# Support

Rackpad is currently maintained as a self-hosted infrastructure project rather
than a managed service.

## Getting help

When filing an issue or asking for help, include:

- Rackpad version or commit
- deployment method (`Docker`, `native Node`, reverse proxy, operating system)
- browser and device, if the issue is UI-related
- relevant logs or screenshots
- whether the issue appears only with demo data, restored backups, or specific
  real-lab data

## Best-effort scope

The project is best suited to:

- homelabs
- workshops
- small offices
- private infrastructure inventories

It is not presented as a managed SaaS offering, and there is no guaranteed SLA.
